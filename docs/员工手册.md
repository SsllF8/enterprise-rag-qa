# 星辰科技有限公司 员工手册

## 第一章 公司简介

星辰科技有限公司成立于2019年，总部位于北京市海淀区中关村科技园，是一家专注于企业级人工智能解决方案的科技公司。公司目前拥有员工320人，其中研发人员占比65%。

公司愿景：用AI技术赋能千行百业，让智能化触手可及。
公司使命：为企业提供可靠、高效、易用的AI产品和解决方案。

核心业务包括：智能客服系统、企业知识管理平台、AI数据标注平台、RAG智能问答引擎。

## 第二章 考勤与休假制度

### 2.1 工作时间

公司实行弹性工作制：
- 标准工作时间为每周一至周五
- 核心工作时间：上午10:00-下午16:00（此时间段内必须在岗）
- 弹性时间段：可自行安排，建议上午9:00-10:00到岗，下午16:00-18:00离岗
- 每周工作时长不少于40小时

### 2.2 年假制度

| 工龄 | 年假天数 |
|------|----------|
| 入职1年以内 | 5天 |
| 入职1-5年 | 7天 |
| 入职5-10年 | 10天 |
| 入职10年以上 | 15天 |

年假可分多次使用，每次不少于半天。当年未使用的年假可延续至次年第一季度，逾期作废。

### 2.3 事假与病假

- 事假：需提前1个工作日提交申请，每月不超过2天，扣除当日薪资
- 病假：凭医院开具的病假条，1天以内无需审批，超过1天需部门经理审批
- 病假薪资：3天以内发放基本工资的100%，3-7天发放80%，7天以上发放60%

### 2.4 加班制度

- 工作日加班：需提前在OA系统提交申请，经审批后方可计算加班
- 加班补偿：可选择调休（1:1）或加班费（平时1.5倍，周末2倍，法定节假日3倍）
- 每月加班时长上限为36小时

## 第三章 薪酬与福利

### 3.1 薪酬结构

月薪 = 基本工资（70%）+ 绩效工资（30%）

- 基本工资：每月固定发放
- 绩效工资：根据季度KPI考核结果发放，系数为0.8-1.2
- 年终奖：通常为2-4个月薪资，根据公司业绩和个人表现确定

### 3.2 五险一金

公司按照北京市上限标准缴纳五险一金：
- 养老保险：个人8%，公司16%
- 医疗保险：个人2%+3元，公司9.8%
- 失业保险：个人0.5%，公司0.5%
- 工伤保险：公司0.4%
- 生育保险：公司0.8%
- 住房公积金：个人12%，公司12%

### 3.3 补充福利

1. 餐饮补贴：每月800元，随工资发放
2. 交通补贴：每月500元，随工资发放
3. 通讯补贴：每月200元，需提供发票报销
4. 年度体检：每年一次，标准为2000元/人
5. 节日福利：春节、中秋、端午各发放1000元礼品卡
6. 团建经费：每人每年3000元
7. 学习基金：每人每年5000元，可用于培训、书籍、在线课程

## 第四章 差旅与报销

### 4.1 出差审批

- 省内出差：部门经理审批
- 省外出差：部门总监审批
- 出差需提前3个工作日在OA系统提交申请
- 出差返回后5个工作日内提交报销

### 4.2 差旅标准

| 项目 | 标准（一线城市） | 标准（其他城市） |
|------|-----------------|-----------------|
| 住宿 | 500元/晚 | 350元/晚 |
| 餐饮 | 150元/天 | 120元/天 |
| 市内交通 | 100元/天 | 80元/天 |

交通工具标准：
- 高铁/动车：二等座
- 飞机：经济舱（4小时以上航程可选）
- 市内交通：优先使用公共交通，打车需说明原因

### 4.3 报销流程

1. 登录OA系统 → 费用报销模块
2. 填写报销单，上传电子发票
3. 等待直属经理审批（2个工作日内）
4. 财务部复核（3个工作日内）
5. 打款至工资卡（次月工资日一并发放）

注意：超过5000元的单笔报销需附部门总监签字。

## 第五章 IT与信息安全

### 5.1 办公设备

- 入职配置：笔记本电脑（MacBook Pro 14寸或ThinkPad X1 Carbon）
- 显示器：可申请外接显示器，标准为27寸4K
- 外设：键盘、鼠标、耳机由公司统一采购

### 5.2 账号管理

- 企业邮箱：name@startech-ai.com
- 企业微信：入职当天开通
- 开发环境：GitLab、Jira、Confluence 账号由IT部门在入职3天内开通
- VPN账号：远程办公需要，需向IT部门申请

### 5.3 信息安全规范

1. 禁止将公司代码上传至个人GitHub等公开仓库
2. 禁止使用未经安全检测的U盘接入公司网络
3. 工作电脑必须安装公司统一的安全软件
4. 敏感数据（客户信息、财务数据）禁止通过个人微信传输
5. 离职时需交回所有办公设备，账号将在离职当天停用
